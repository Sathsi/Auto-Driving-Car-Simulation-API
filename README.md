# Auto-Driving-Car-Simulation
Auto Driving Car Simulation

## Content
- Build the project
- Running the app
- End Points

### Build the project
```
   mvn clean install
```

### Running the app
java -Dserver.port=8032 -jar automate-car-0.1.jar

## End Points

1. Once deployed the app, user can see the swagger API documentation.
```
    http://localhost:8032/swagger-ui.html#/
```
